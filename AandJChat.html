After this, we'll be unable to control the Rift in any way ever again.

AM: And would that close the Rift permanently?

J: Don't think so. It seems whatever caused the anomaly in the first place left some permanent damage but we can help to heal it, even if just slightly.

AM: Alright. Do you have plans for how we're to actually do this?

J: I have The Mechanic sketching up some blueprints right now. Should be done some time soon.
